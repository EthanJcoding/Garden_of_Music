```
악보 판매 등록부터 연주 영상 공유까지!
악보 거래 플랫폼, 악보의 정원
```

<b>악보의 정원</b>은 실력 향상과 새로운 기술 스택 적용을 위해 모인 프론트엔드 개발자 3명이 함께하는 팀 프로젝트입니다.
<br>
TypeScript와 React를 기반으로 하며 Atomic Design Pattern을 적용한 UI 개발을 시도하고 있습니다.🤔
